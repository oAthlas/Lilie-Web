# Lilie Web - A continuação da Assistente Virtual de Bolso

A Lilie original era um projeto feito para a feira de ciências do colégio, o projeto visava extrema portabilidade e conexão multidispositivos. Em sua nova versão a Lilie agora está disponivel em uma versão web que pode ser acessado através de https://lilie-web.onrender.com.

## Objetivo

A Lilie agora é um chatbot simples, sem todas as funcionalidades extras e ultra conectividade, sendo um simples chatbot que responde perguntas simples com interface amigável e intuitiva.

## Novidades

- Interface nova usando tons pasteis de vermelho e rosa
- Perca de personalidade em troca de extremma abrangência de tons a serem explorados
- Extrema simplicidade, focando completamente no aprendizado enquanto construída
- Novo modelo de linguagem, antes DeepSeek via OpenRouter, agora Gemini via GenAi
